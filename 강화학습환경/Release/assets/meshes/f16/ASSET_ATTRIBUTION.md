- The viewer accepts any local OBJ/VTK-readable F-16 mesh via `--mesh`.

  Sketchfab login or a separate download step.
- If a higher fidelity external CC-BY asset is later downloaded, place it in
  this directory and update this attribution file with the original author,
  source URL, license, and modification notes.
