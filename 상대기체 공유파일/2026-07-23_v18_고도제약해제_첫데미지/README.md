# v18 — 고도 제약 해제 (사상 첫 데미지)

vs 권정환 BT 200초 1경기
```
total_reward   : +1618.92      (v17 +242.1 / v16 -118 / v15 -21.1)
ownship_health : 1.0           무손상
target_health  : 0.8429        ★ v0~v17 통틀어 처음으로 상대에게 입힌 피해
end_condition  : max time out
```

## 무엇이 문제였나 (실측으로 규명)

`ata_split.py`로 ATA를 수평/수직 성분으로 분해하니 원인이 드러났다.

```
v17 사거리 내 : 수평오차 5.86deg / 수직오차 22.88deg
                VP Z클램프 발동 93.3%,  고도차 -274m (상대가 계속 아래)
```

`alt_trace.py`로 확인한 고도 추이:
```
t=20s 이후 200초 내내 우리 고도 3358~3492m 에 고정
상대는 3066m, 최저 2160m 까지 자유롭게 하강
```

범인은 `Task_LeadPredict.cpp` 맨 마지막 줄이었다.
```cpp
if (predicted.Z < minZ) predicted.Z = minZ;       // diveSlope
if (predicted.Z > maxZ) predicted.Z = maxZ;
if (predicted.Z < 3500.0) predicted.Z = 3500.0;   // ★마지막 = 최종 결정권
```
**우리는 3500m 아래로 내려갈 수 없게 스스로 묶여 있었다.** 그래서 사거리 안에
87초를 머물러도 상대를 내려다보기만 했고, 데미지가 0이었다.
실제 종료 하한은 `config.py: min_altitude = 300.0` — **11.7배 과보수적**이었다.
ClimbOut(MinAlt 3000)도 그 하한 위라 200초간 0회 발동하는 죽은 분기였다.

## 수정 (4건, 모두 "고도 제약" 한 축)

| # | 수정 | 근거 |
|---|---|---|
| 1 | VP Z 절대하한 `3500 -> 1500m` | 80% 틱에서 발동. 종료(300m)까지 1200m 여유 |
| 2 | `diveSlope dist*0.2 -> *0.5` | 하강 조준각이 11.3deg로 제한됨(상승은 26.6deg) |
| 3 | ClimbOut `MinAlt 3000 -> 1800` | 하한보다 위에서 발동해 안전망으로 부활 |
| 4 | 에피소드 경계 static 초기화 | `fwdHist`/`lastThr`가 판을 넘어 잔존(배치 신뢰성) |

2번의 급강하 가속(26.6deg에서 4.4m/s^2)은 v14 dV 폐루프의 스로틀 여유
(1.0->0.55 = 약 4.5m/s^2)로 상쇄 가능하다고 보고 감수했다.

## 결과 검증 (점수가 아니라 원인 지표로)

| 지표 | v17 | v18 |
|---|---|---|
| 수직 오차 | 22.88deg | **5.06deg** |
| 수평 오차 | 5.86deg | 4.42deg |
| Z클램프 발동 | 93.3% | **0.2%** |
| 고도차 | -274m | **+78m** |
| 사거리 체류 | 87.5초 | **143.6초** |
| 사거리 내 최소 ATA | 13.33deg | **0.17deg** |
| 사격조건(ATA<=1) 도달 | 0틱 | **33틱** |

수정한 축(수직)만 정확히 움직였고 수평은 거의 그대로다 = 원인 규명이 맞았다.

**안전성:** 우리 최저 고도 2345m. 새 하한(1500m)과 ClimbOut(1800m) **모두 0회 발동**.
여유를 두고 내려간 것이지 위험하게 내려간 것이 아니다.

## 남은 문제 — 거리대별 조준 격차

```
600~900m : ATA 0.2~0.8deg  (데미지계수 0.22~0.35)  <- 여기서만 사격 성립
152~500m : ATA 18~37deg    (데미지계수 0.61~1.00)  <- 계수는 크나 조준 불가
```
근접할수록 각속도 요구가 폭증해 조준이 무너진다. 데미지 계수는 반대로 근접이 크다.
현재는 조준이 되는 600~900m에서만 득점 중이며, 33틱(0.55초)에 그친다.
다음 과제는 **그 사격 상태를 오래 유지하는 것**.
